# Automação de Testes com Playwright

Projeto desenvolvido para prática de testes automatizados E2E.

## Cenários testados
- Criação de tarefa
- Validação de tarefa duplicada
- Campo obrigatório

## Tecnologias
- Playwright
- TypeScript

## Como executar
npm install
npx playwright test
