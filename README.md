# Automação de Testes com Playwright

Projeto desenvolvido para prática de testes automatizados E2E.

## Cenários testados
- Criação de tarefa
- Validação de tarefa duplicada
- Campo obrigatório

## Tecnologias
- Playwright
- TypeScript

## Como executar
npm install
npx playwright test

## Sobre o projeto

Este projeto foi desenvolvido durante meus estudos em automação de testes,
onde apliquei conceitos como Page Object Model e validação de cenários reais.
